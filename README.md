# Store
RDNSTORE adalah platform toko online (e-commerce) premium berkonsep Single Page Application (SPA) yang khusus menjual perangkat teknologi bekas berkualitas tinggi (Premium Secondhand Tech). Website ini dimiliki oleh Raden Antasena Kartasasmita dan berfokus pada penjualan laptop serta komputer (PC) rakitan dengan transparansi kondisi unit 100%.
